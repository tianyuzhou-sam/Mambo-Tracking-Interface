For NGV-02 (Windows)

For ROAHM LAB XPS 1 (Linux)
1. Turn on Bluetooth
2. Connect Wi-Fi to NETGEAR48
3. Open 1 Terminal, type <cd Mambo_scripts/low_level_controller/scripts/>
4. Press <Crtl> + <Shift> + <T> to create Terminal 2, the directory should be same with Step 3
4. type <python3 mocap.py> in Terminal 1
5. type <python3 LLC.py> in Terminal 2
6. After you see "Total Time" in Terminal 2, wait for 1 second, run the MATLAB planner.
